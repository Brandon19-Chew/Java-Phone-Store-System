# Java-Phone-Store-System

Develop the system by using JAVA code language. The purpose of develop the system is have an online program for the staffs to do the management and users to buy the products. 
Components module = Login / Logout, Account Creation, Topup Money, Menu Selection, Generate Invoice (I), Cart Items Display, Login / Logout History, Payment, Items Records, Purchase Order (PO),                          Restock Items and Error Tracker.
